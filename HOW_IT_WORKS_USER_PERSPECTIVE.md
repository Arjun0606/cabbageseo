### Risk 1: Empty Dashboard

**Solution:** Auto-scan always runs, never show empty state

**Problem:** User sees problem, doesn't know what to do

**Problem:** User sees steps, feels overwhelmed
**Solution:** Micro-wins, progress tracking, clear outcomes

**Problem:** User does steps, sees no improvement
**Solution:** Track progress, show week-over-week changes
